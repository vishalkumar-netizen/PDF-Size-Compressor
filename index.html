How to Use This Tool

  1. Upload File: Drag and drop your PDF or click the dashed area to browse. (No size limits).
  2. Select Mode:
    • Desired Size: Enter the exact target file size you need (e.g., 500 KB or 2 MB).
      Note: If you request a massive reduction, the tool prioritizes file size over visual quality.
    • Percentage: Reduce the file size by a relative slider ratio.
    • Smart Auto: The safest method. It cleans up hidden metadata and bloat while keeping 100% of the original visual quality.
  3. Compress: Click the "Compress PDF" button. Everything happens securely inside your browser; no data is sent to a server.
  4. Save: Preview your optimized PDF or download it directly to your device.
